CSS

CSS (Cascading Style Sheets) is a style sheet language used to design and control the appearance of web pages. It allows developers to style text, layouts, colors, backgrounds, and animations separately from the HTML structure, making websites visually appealing and consistent.
  • 3 Main Features of CSS
    1. Styling Web Pages
      2. Control fonts, colors, spacing, borders, backgrounds, and layout of HTML elements.
    2. Separation of Content and Design
      3. Keeps HTML for structure and CSS for appearance, improving maintainability.
    3. Responsive Design
      4. Supports media queries to make websites adapt to different screen sizes like mobiles, tablets, and desktops.
  • Basic CSS Workflow
    1. Add CSS
      2. Apply styles to HTML using inline, internal, or external CSS.
    2. Select Elements
      3. Use selectors like element, class, ID, or universal to target HTML elements.
    3. Apply Properties
      4. Style elements using properties like color, font-size, margin, padding, background, etc.
    4. Test and Adjust
      5. Preview changes in the browser and tweak properties for desired appearance.
    5. Make Responsive
      6. Use media queries to adjust styles for different devices and screen sizes.
  • CSS Basic Properties
    1. Color
      2. Sets text color. Example: color: red;
    2. Font-size
      3. Sets the size of text. Example: font-size: 16px;
    3. Font-family
      4. Sets font type. Example: font-family: Arial, sans-serif;
    4. Text allign
      5. Aligns text: left, right, center, justify. Example: text-align: center;
    5. Background Color
      6. Sets background color of an element. Example: background-color: lightblue;
    6. Background Image
      7. Adds an image as background. Example: background-image: url('image.jpg');
    7. Width / height
      8. Sets element dimensions. Example: width: 300px; height: 150px;
    8. Margin / padding
      9. Sets space outside and inside elements. Example: margin: 10px; padding: 5px;
    9. Border / Border-radius
      10. Adds border and rounds corners. Example: border: 1px solid black; border-radius: 5px;
    10. Display
      11. Controls element layout: block, inline, flex, none. Example: display: flex;
