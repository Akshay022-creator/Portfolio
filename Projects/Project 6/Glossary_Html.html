HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the skeleton of a webpage by organizing text, images, links, and other elements using tags and attributes.
  • Features of HTML
Structures webpages using headings, paragraphs, lists, tables, etc.
Supports images, videos, and multimedia.
Works with CSS (styling) and JavaScript (interactivity).
Uses hyperlinks to connect pages.
  • Basic Structure Tags
    1. DOCTYPE html
      2. Defines the document type and version of html
    2. head
      3. Contains metadata, title, links, and scripts
    3. title
      4. Sets the webpage title shown in the browser tab
    4. body
      5. Contains visible content of the webpage
  • Types of Lists
    1. UL
      2. Unordered list (bullets)
    2. OL
      3. Ordered list (numbers)
    3. LI
      4. List item
    4. DL
      5. Description list
    5. DT
      6. Term in description list
    6. DT
      7. Description of term
